# **App Name**: Sofia Nails

## Core Features:

- Appointment Creation: Allow clients to create appointments, providing necessary details such as service type, date, and time.
- Admin Authentication: Secure the admin view with authentication to ensure only authorized personnel can access appointment data.
- Appointment Management: Enable the admin to view, modify, and delete appointments.
- Data Storage: Use a database (such as a Cloud SQL database) to store and manage appointment data for persistent access.

## Style Guidelines:

- Primary color: Soft pink (#F4B4C3) to evoke a sense of femininity and beauty.
- Background color: Light pink (#F9D9E2), slightly desaturated, provides a gentle backdrop.
- Accent color: Lavender (#D0A2F7), offering a contrast that's still harmonious and calming.
- Font pairing: 'Belleza' (sans-serif) for headings to convey a fashionable and artistic aesthetic, paired with 'Alegreya' (serif) for body text to provide readability.
- Use delicate, custom-designed icons related to nail care, such as nail polish bottles, brushes, and nail shapes.
- Clean and intuitive layout with clear sections for appointment creation and admin management. Use a grid system for responsiveness.
- Subtle transitions and animations to enhance user experience, such as fading in new appointments or smoothly sliding panels for admin controls.
